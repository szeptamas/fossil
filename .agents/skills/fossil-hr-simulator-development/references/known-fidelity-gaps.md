# Known fidelity gaps

- Browser JavaScript is not JerryScript v2.1.0.
- Browser memory and timing are not watch memory and timing.
- Fonts and text metrics differ from firmware.
- Icon rendering is not implemented.
- E-paper refresh and ghosting are not simulated.
- Physical circular clipping is approximate.
- Hand position is visual, not mechanical.
- Timers use browser timers.
- Persistence uses browser local storage.
- Phone requests are logged but not fulfilled.
- Sensor data is synthetic.
- Generic notifications are logged rather than displayed as firmware overlays.
